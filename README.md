# Dependencies
## FrontEnd Dependencies
### Selalu pastikan dalam directory client 
cd client
### Install
- npm install react-router-dom jwt-decode
- npm install -D tailwindcss postcss autoprefixer

### Run Dev Client
- npm run dev
## BackEnd Dependencies
### Selalu pastikan dalam directory server 
cd server

### Install
- npm install express firebase-admin mqtt dotenv cors bcrypt jsonwebtoken express-rate-limit
- npm install --save-dev nodemon

### Run Dev Server
- npm run dev
